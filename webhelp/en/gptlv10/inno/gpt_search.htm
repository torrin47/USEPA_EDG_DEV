Using Geoportal Extension Search
Release 9.3.1

Introduction

The Geoportal extension search is a powerful way to discover metadata documents leading to valuable data resources. The following sections describe how to use Geoportal extension search functionality to its fullest.



You can also search a Geoportal from the browser directly by adding it to your list of search providers.

  • There are a number of search page options available to help narrow search results.
  • Users can leverage lucene search query syntax
    • to refine search terms.
  • Users can also search through an ontology service in the Geoportal to refine search terms.
  • Search results can be viewed and used in many different ways.
