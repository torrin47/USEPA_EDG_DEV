Intranet Link Explanation


Some content of this site is intended for EPA Intranet users only. If you are accessing this site from within EPA's telecommunications network, links marked will work. If you are an EPA business partner or affiliate who has business with the EPA, please use the Feedback link to request access to all GDG content. To learn more about logging into the GDG, please visit EPA's GeoData Gateway Login Help page.







The EDG is EPA's central environmental metadata portal. Please read EPA's Privacy and Security Notice.
