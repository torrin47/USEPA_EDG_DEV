EPA Examples Using the REST API

The GDG's REST API provides you with a powerful way to access and embed GDG content directly in other web applications or pages. The GDG REST interface is available through both internet and intranet access points. The intranet URL provides users with access to the full set of EPA records (restricted and unrestricted) without requiring that users login, while the internet URL only provides those users who are logged into the GDG with access to restricted records. Only those users who can get to EPA's intranet can access the intranet URL. Those who cannot access EPA's intranet must use the internet URL.

The following information is intended to illustrate EPA-specific examples of using the GDG's REST interface. For each example provided below, an internet and intranet URL is provided. For more information on all REST parameters, please see the REST details page.

  • Facility search output in GeoRSS format (internet): https://geogateway.epa.gov/geoportal/rest/find/document?searchText=facilities&f=georss&xsl=metadata_to_html_full

  • Facility search output in GeoRSS format (intranet): http://geogateway-intranet.epa.gov/geoportal/rest/find/document?searchText=facilities&f=georss&xsl=metadata_to_html_full

  • Facility search output in GeoRSS format with the content type (live data) specified (internet): https://geogateway.epa.gov/geoportal/rest/find/document?searchText=facilities&f=georss&contentType=liveData&xsl=metadata_to_html_full
  • Facility search output in GeoRSS format with the content type (live data) specified (intranet): http://geogateway-intranet.epa.gov/geoportal/rest/find/document?searchText=facilities&f=georss&contentType=liveData&xsl=metadata_to_html_full
  • Facility search with html output (internet): https://geogateway.epa.gov/geoportal/rest/find/document?searchText=facilities&f=html&xsl=metadata_to_html_full
  • Facility search with html output (intranet): http://geogateway-intranet.epa.gov/geoportal/rest/find/document?searchText=facilities&f=html&xsl=metadata_to_html_full
  • Facility search with html output using a stylesheet (internet): https://geogateway.epa.gov/geoportal/rest/find/document?searchText=air+quality&f=html&xsl=metadata_to_html_full&style=https://geogateway.epa.gov/geoportal/webhelp/en/geoportal_931/rest.css
  • Facility search with html output using a stylesheet (intranet): http://geogateway-intranet.epa.gov/geoportal/rest/find/document?searchText=air+quality&f=html&xsl=metadata_to_html_full&style=https://geogateway.epa.gov/geoportal/webhelp/en/geoportal_931/rest.css
  • Search for term "ecoregions" with KML output (internet): https://geogateway.epa.gov/geoportal/rest/find/document?searchText=ecoregions&f=kml&xsl=metadata_to_html_full
  • Search for term "ecoregions" with KML output (intranet): http://geogateway-intranet.epa.gov/geoportal/rest/find/document?searchText=ecoregions&f=kml&xsl=metadata_to_html_full
  • Results can also be embedded into a web page using the iframe tag: iframe width="500" height="200" src="https://geogateway.epa.gov/geoportal/rest/find/document?searchText=toxic+release&f=html&xsl=metadata_to_html_full&style=https://geogateway.epa.gov/geoportal/webhelp/en/geoportal_931/rest.css"
