Introduction to Geoportal Extension REST API

The purpose of the Geoportal extension 9.3.1 REST interface is to provide an easy-to-use URL-based interface for querying a metadata catalog. By providing this functionality in such an interoperable and standard way, the Geoportal catalog can be opened up to a vast array of integration possibilities with other web application environments.
