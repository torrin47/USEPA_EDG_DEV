How to Use the Data Download Feature

If the Data Download feature is available for this Geoportal (a "Download" tab will be visible in the Geoportal interface), then you will be able to download certain data through the Data Download interface. The data you select will be emailed to you in a zipped file.



To download data follow these steps:

  1. Select the layers of data that you want to download from the list to the right of the interactive map.
  2. Specify the email address that should receive the zipped data.
  3. Select an output file format from the available file formats in the dropdown list.
  4. Select the projection from the available map projections in the dropdown list.
  5. Draw an extent using the interactive map. First click on the map to activate it, then use the tools to define the extent:
    • Zoom In/Out: There are three ways to zoom in. One way is to use the up and down arrow buttons on the upper left corner of the map. Another way is to use the "+" or "-" keys on the number pad of your keyboard. The third way is to use the scroll bar on your mouse.
    • Draw Polygon: Click the Draw Polygon button, and then click on the map and drag a square around to create a bounding box for the spatial extent you want to download. Release the mouse when the box is the desired extent.
  6. Click the download button to submit a request for downloadable data. A dialog will appear asking if you are sure that you want the data sent. Confirm, and then check your email to receive the file.
