Welcome to the Environmental Dataset Gateway (EDG)

Two Types of Help

There are two types of help for the EDG. You can click on the links below or use the navigation at the tab to the left to move through help topics.

Tip:

In addition, each geoportal web application hosts context-sensitive help. Context-sensitive help means that if you need help while working with a geoportal, you can click the help link at the top of the page and a window with guidance specific to that page will open.

  1. User Help: This is the place to get help for EDG functionality, particularly for people who use geoportals but do not install or customize them. The links below can get you started.
    • What is a geoportal and the EDG
    • What's New in the Geoportal Server
    • User Functions and Roles
    • How to Search for Resources
    • How to Publish Resources
    • How to Manage and Edit Resources
    • Geoportal Clients for ArcGIS
  2. Developer Help: This is the place to get help for geoportal management considerations, installation, customizations, and support for staff maintaining a geoportal implementation. The links below can get you started. Additional resources and customizations are available at the Geoportal Resource Center.
    • Management Considerations
    • Installation
    • Customizations
    • Getting Assistance with a Geoportal Implementation
